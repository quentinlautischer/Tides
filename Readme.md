# Tides


## How to run
1. python3 -m venv TidesEnv
2. source TidesEnv/bin/activate
3. [When finished] deactivate

## Info
We are pulling data from: https://www.tides.gc.ca/en/stations/7735 for Vancouver. If you want a different location you will need to download a different bulk batch of data from said location.

## Key Features
- Provide date and time for Next N number below X level
- Select Location
- FindTide(N, X, location=Vancouver)
